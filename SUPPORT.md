# Support

Thanks for using forgejo-mcp. Here is where to go for help.

## Questions and usage help

- Read the [README](README.md) first: it covers configuration
  (`FORGEJO_BASE_URL`, `FORGEJO_TOKEN`), the tool list, and the client setup
  snippet.
- For "how do I..." questions and general discussion, open a
  [GitHub issue](https://github.com/rubicon/forgejo-mcp/issues) with the
  `question` label.

## Bug reports and feature requests

- Open a [GitHub issue](https://github.com/rubicon/forgejo-mcp/issues). Include
  the server version, your Node version, the tool you called, the arguments (with
  secrets redacted), and what you expected versus what happened.
- Please do not paste tokens, real base URLs, or private repository data into an
  issue.

## Security issues

Do not report security problems in public issues. Follow the
[Security Policy](SECURITY.md) for private disclosure.

## Support expectations

This is an open-source project maintained on a best-effort basis. There is no
service-level guarantee. Well-scoped issues and pull requests that follow
[CONTRIBUTING.md](CONTRIBUTING.md) get the fastest response.
